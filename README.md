# AskDevi-Assignment

## Installation :

### 1. Clone the repo
```bash
git clone https://github.com/Hrish17/AskDevi-Assignment.git
cd AskDevi-Assignment
```

### Frontend :
```bash
python3 -m venv venv
source venv/bin/activate


cd backend
pip install -r requirements.txt


python manage.py migrate
python manage.py runserver
```
